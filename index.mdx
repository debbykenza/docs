## À qui s’adresse cette API ?

1. Développeurs d’applications mobiles ou web souhaitant intégrer un système de livraison.
2. Équipes techniques internes cherchant à automatiser les processus logistiques.
3. Partenaires tiers souhaitant s’interfacer avec DOM.
