## Installation de l’interface en ligne de commande (CLI)

<Info>
  Prérequis : veuillez installer Node.js avant de continuer.
</Info>

<Steps>
  <Step title="Install the CLI.">
    Run the following command to install the [CLI](https://www.npmjs.com/package/mint):

    ```bash
    npm i -g mint
    ```
  </Step>
  <Step title="Prévisualiser en local.">
    Accédez à votre répertoire de documentation (là où se trouve votre fichier`docs.json`) et exécutez la commande suivante:

    ```bash
    mint dev
    ```
  </Step>
</Steps>

Une prévisualisation locale de votre documentation sera disponible à l’adresse`http://localhost:3000`.

Sinon, si vous ne souhaitez pas installer la CLI globalement, vous pouvez exécuter un script unique :

```bash
npx mint dev
```
